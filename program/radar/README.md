# 软媒系统雷达

官方下载地址：<https://mofang.ruanmei.com/>

## 开机启动

- 双击`_开机启动.bat`

## 解决使用电池无法启动问题

- 双击`_打开任务计划程序.bat`，打开`任务计划程序`
- 选择`任务计划程序库`
- 双击`radar`
- 选择`条件`
- 取消勾选`电源`里的2个勾
